## Technologies
Project is created with:
* Python 3.9
  * Modules:
      *  Scipy
      *  numpy
      *  matplotlib



## Setup
To run this project clone the repository and install it using pip:

```
cd PredatorPreyModel
pip install -r requirements.txt
python main.py
```
